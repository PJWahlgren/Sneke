## Install
> [!NOTE]
> The project was done on Linux so no guarantees it works anywhere else.

Make sure raylib is installed (see instructions how [here](https://github.com/raysan5/raylib/wiki/Working-on-GNU-Linux))

Then to run the game write
```make run```

## Cool concepts used and learned
* Makefile
* No malloc, just stack allocation
* MVC
* Common interface to the view by using a char array buffer
* Raw data pointer
* Raylib
